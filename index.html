Welcome to OpenHMD.net

About

OpenHMD aims to provide a Free and Open Source API and drivers for immersive technology, such as head mounted displays with built in head tracking.

Devices

The only supported device at the moment is the Oculus Rift SDK.

Operating Systems

  • Linux
  • OS X
  • Windows

The project is hosted on github, where the source code, downloads and usage examples are available.

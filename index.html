Welcome to OpenHMD.net

About

OpenHMD aims to provide a Free and Open Source API and drivers for immersive technology, such as head mounted displays with built in head tracking. Our aim is to implement support for as many devices as possible in a portable, cross-platform package.

Devices

  • Oculus Rift DK1
  • Oculus Rift DK2 (rotation only)
  • Oculus Rift CV1 (rotation only, git version only)
  • HTC Vive (rotation only, currently in branch)
  • Generic Android framework (with specific profiles for devices like the Durovis Dive, Zeiss VR One, Google Cardboard to name a few, or even custom build android devices are in development.
  • External sensor data - Pass through your own data and use the OpenHMD framework and sensor fusion!

Operating Systems

  • Android
  • FreeBSD
  • Linux
  • OS X
  • Windows

Source Code & Downloads

The project is hosted on github, where the source code, downloads and usage examples are available.

Pre build packages are also known to exist in the Arch Linux repository and the Mac OSX Homebrew repository. The perl bindings are also available on CPAN
